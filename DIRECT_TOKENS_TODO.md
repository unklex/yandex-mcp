# Яндекс.Директ — настройка токенов (не завершено)

> Этот файл можно удалить после успешной настройки.

## Текущий статус

| Аккаунт | Счётчик Метрики | Direct-токен | Статус |
|---------|----------------|--------------|--------|
| `promreo` | YOUR_COUNTER_ID_1 | ❌ временный (токен Метрики) | **Нужен отдельный** |
| `site2` | YOUR_COUNTER_ID_2 | ❌ временный (токен Метрики) | **Нужен отдельный** |

Оба аккаунта сейчас используют токен Яндекс.Метрики как заглушку.
Это **не будет работать** для Яндекс.Директа — нужны токены с доступом к Директу.

## Как получить токены для Директа

Один токен подойдёт для обоих аккаунтов **только если** оба Директ-аккаунта
привязаны к одному Яндекс-логину. Если это разные логины — нужны разные токены.

### Шаг 1. Убедитесь, что приложение имеет доступ к Директу

Откройте [oauth.yandex.com](https://oauth.yandex.com) → ваше приложение →
убедитесь, что в разделе **Доступы** отмечено:
- ✅ `direct:api` — Использование API Яндекс.Директа

Если нет — добавьте разрешение и сохраните приложение.

### Шаг 2. Получите токен

**Для каждого аккаунта** (залогиньтесь под нужным Яндекс-логином):
```
https://oauth.yandex.com/authorize?response_type=token&client_id=YOUR_CLIENT_ID
```

Скопируйте `access_token=...` из адресной строки после авторизации.

Если аккаунты на разных логинах — откройте в разных браузерах или в режиме инкогнито.

### Шаг 3. Обновите `.env`

Замените токены-заглушки на реальные:

```env
YANDEX_DIRECT_ACCOUNTS=promreo:TOKEN_ДЛЯ_PROMREO,site2:TOKEN_ДЛЯ_SITE2
```

Если оба аккаунта на одном логине — токены будут одинаковые.

### Шаг 4. Перерегистрируйте MCP-сервер

```cmd
claude mcp remove yandex-metrica
claude mcp add --transport stdio yandex-metrica ^
  --env YANDEX_METRICA_TOKEN=YOUR_METRICA_TOKEN ^
  --env YANDEX_METRICA_COUNTER_ID=YOUR_COUNTER_ID ^
  --env YANDEX_METRICA_COUNTERS=promreo:COUNTER_1,site2:COUNTER_2 ^
  --env YANDEX_DIRECT_ACCOUNTS=promreo:DIRECT_TOKEN_1,site2:DIRECT_TOKEN_2 ^
  -- python C:/Projects/claude-mcp/yametrika/server.py
```

## Проверка после настройки

В Claude Cowork напишите:
```
Покажи список кампаний Директа для аккаунта promreo
```

Если ответ содержит список кампаний — всё работает.
Если ошибка 401 — токен неверный или не имеет доступа к Директу.
